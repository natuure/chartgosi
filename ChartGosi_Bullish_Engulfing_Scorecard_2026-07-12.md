# 차트고시 상승장악형(Bullish Engulfing) 스코어카드

작성일: 2026-07-12

## 일러두기

아래 "몸통 장악 완성도"·"장악 강도" 항목(둘째 날 양봉이 첫째 날 음봉의 몸통을 가격상 완전히
감싸야 한다는 조건)은 사용자가 준 정의에 명시적으로 들어있지 않지만, "장악형(Engulfing)"이라는
패턴 이름 자체가 이 조건을 전제로 하므로 표준 정의를 따라 추가했다. 불필요하면 해당 항목만
제거하고 점수를 다른 항목에 재배분하면 된다. 그 외 캔들 순서, 꼬리 비율 계산식, 위치 조건(52주
신고가 대비 하락률), 저점권 조건은 전부 사용자가 준 정의를 그대로 반영했다.

## 기본 관점

상승장악형은 **일봉 기준 이틀 연속 두 봉으로 완성되는 반전형 캔들 패턴**으로 본다.

핵심 전제:

- 첫째 날은 음봉, 둘째 날은 양봉이어야 한다.
- 두 봉 모두 윗꼬리와 아래꼬리가 없거나 매우 적어야 한다(몸통 위주의 캔들일수록 신뢰도가 높다).
- 둘째 날 양봉의 몸통이 첫째 날 음봉의 몸통을 가격상 완전히 감싸야 한다.
- 패턴은 52주(약 252거래일) 신고가 대비 20% 이상 하락한 위치에서 발생해야 한다.
- 상승장악형은 추세전환봉이므로, 음봉과 양봉 각각의 저가가 해당 봉 앞뒤 10개봉 저가 비교에서
  하위 3개봉 안에 들어야 한다.
- 문제 화면은 음봉, 양봉 장악봉, 양봉 바로 다음 봉까지만 보여준다.
- 정답 판정용 다음 5봉은 양봉 바로 다음 봉 이후부터 계산한다.

## 꼬리 비율 계산 정의

음봉일 때:

```text
윗꼬리 비율  = (고가 - 시가) / (고가 - 저가)
아래꼬리 비율 = (종가 - 저가) / (고가 - 저가)
```

양봉일 때:

```text
윗꼬리 비율  = (고가 - 종가) / (고가 - 저가)
아래꼬리 비율 = (시가 - 저가) / (고가 - 저가)
```

고가와 저가가 같아(고가-저가 = 0) 비율 계산이 불가능한 봉이 있으면 후보에서 제외한다.

## 필수 구조

1. 첫째 날 = 음봉(시가 > 종가), 둘째 날 = 양봉(종가 > 시가).
2. 둘째 날(양봉) 시가 ≤ 첫째 날(음봉) 종가, 둘째 날(양봉) 종가 ≥ 첫째 날(음봉) 시가 — 양봉
   몸통이 음봉 몸통을 가격상 완전히 감싼다.
3. 패턴 형성일(음봉 시작일) 기준 최근 252거래일 최고 종가 대비 하락률이 20% 이상이어야 한다.
4. 패턴 완성일(양봉) 바로 다음 봉 종가가 음봉·양봉 두 봉의 종가 중 더 낮은 종가 아래로 내려가면 안 된다.
5. 음봉과 양봉 각각의 저가가 해당 봉 앞뒤 10개봉 저가 비교에서 하위 3개봉 안에 들어야 한다.

## 기준 점수

- 85점 이상: 상승장악형 고신뢰 후보
- 75~84점: 상승장악형 후보
- 60~74점: 유사 패턴 가능성이 있어 보류
- 60점 미만: 상승장악형으로 보기 어려움

## 계산 방식

### 1. 몸통 장악 완성도

```text
음봉 몸통크기 = 음봉 시가 - 음봉 종가
하단 장악 여유율 = (음봉 종가 - 양봉 시가) / 음봉 몸통크기
상단 장악 여유율 = (양봉 종가 - 음봉 시가) / 음봉 몸통크기
장악 여유율 평균 = (하단 장악 여유율 + 상단 장악 여유율) / 2
```

두 여유율 모두 0 이상이어야 통과(필수). 값이 클수록 양봉이 음봉을 더 여유 있게 감쌌다는 뜻이다.

- 장악 여유율 평균 30% 이상: 25점
- 15~30%: 18점
- 0~15%: 12점
- 0 미만(하단 또는 상단 중 하나라도 음수): 필수 탈락

### 2. 장악 강도(몸통 크기 비율)

```text
양봉 몸통크기 = 양봉 종가 - 양봉 시가
몸통 비율 = 양봉 몸통크기 / 음봉 몸통크기
```

- 2.0배 이상: 15점
- 1.5~2.0배: 11점
- 1.0~1.5배: 7점
- 1.0배 미만: 필수 탈락(장악 실패와 사실상 동일한 약한 신호로 판단)

### 3. 위치 조건(52주 신고가 대비 하락률)

```text
52주 신고가 = 음봉 시작일 이전 252거래일 중 최고 종가
하락률 = (52주 신고가 - 음봉 종가) / 52주 신고가
```

- 40% 이상: 20점
- 30~40%: 16점
- 20~30%: 12점
- 20% 미만: 필수 탈락

### 4. 음봉 / 양봉 꼬리 품질

```text
음봉 꼬리 합 = 음봉 윗꼬리 비율 + 음봉 아래꼬리 비율
양봉 꼬리 합 = 양봉 윗꼬리 비율 + 양봉 아래꼬리 비율
```

각각 아래 기준으로 채점한다(음봉·양봉 동일 기준, 각 20점):

- 10% 이하: 20점
- 10~20%: 14점
- 20~30%: 8점
- 30% 초과: 필수 탈락

### 5. 다음 봉 확인 조건

```text
확인 기준 종가 = min(음봉 종가, 양봉 종가)
```

패턴 완성일(양봉) 바로 다음 봉의 종가가 확인 기준 종가보다 낮아지면 후보에서 제외한다.
문제 화면은 이 다음 봉까지만 보여주며, 정답 판정용 다음 5봉은 그 이후부터 계산한다.
이 조건은 점수화하지 않고 필수 탈락 조건으로만 사용한다.

### 6. 저점권 조건

```text
음봉 저가 순위 = 음봉 기준 앞 10봉 + 음봉 + 뒤 10봉의 저가를 낮은 순으로 정렬했을 때의 순위
양봉 저가 순위 = 양봉 기준 앞 10봉 + 양봉 + 뒤 10봉의 저가를 낮은 순으로 정렬했을 때의 순위
```

음봉과 양봉 모두 저가 순위가 **3위 이내**여야 한다.  
둘 중 하나라도 4위 이하이면 추세전환봉의 저점권 조건을 충족하지 못한 것으로 보고 후보에서 제외한다.

## 문제 화면 표시 근거(제안)

- 음봉/양봉 각각의 시가·고가·저가·종가
- 음봉/양봉 각각의 앞뒤 10개봉 저가 순위
- 몸통 장악 여유율(하단/상단)
- 몸통 크기 비율
- 52주 신고가 대비 하락률
- 음봉/양봉 꼬리 비율(윗꼬리/아래꼬리)
- 양봉 바로 다음 봉 종가와 확인 기준 종가의 차이
