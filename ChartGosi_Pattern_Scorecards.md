# 차트고시 패턴 스코어카드

작성일: 2026-07-04

## 목적

패턴을 “조건 하나가 맞으면 확정”하는 방식이 아니라, 여러 구조적 근거를 점수화해 일정 점수 이상일 때 후보로 분류한다.

이 방식의 목표는 다음과 같다.

- 문제 품질을 일관되게 관리한다.
- 사용자가 “왜 이 패턴인지” 납득할 수 있게 한다.
- 향후 자동 패턴 탐지에서 룰 기반 점수로 문제 후보를 필터링한다.
- AI 리포트가 사용자의 취약 패턴을 설명할 때 같은 기준을 재사용한다.

## 공통 판정 방식

- 각 패턴은 100점 만점 scorecard를 가진다.
- `primary_threshold` 이상이면 해당 패턴 후보로 분류한다.
- `high_confidence_threshold` 이상이면 고신뢰 후보로 본다.
- 무효 조건은 감점으로 처리한다.
- 점수는 문제 생성/검수용 기준이며, 투자 판단 기준이 아니다.

## 1. 컵앤핸들 스코어카드

### 기준 점수

- 85점 이상: 컵앤핸들 고신뢰 후보
- 75~84점: 컵앤핸들 후보
- 60~74점: 유사 패턴 가능성이 있어 보류
- 60점 미만: 컵앤핸들로 보기 어려움

### 가점 항목

| 항목 | 점수 | 설명 |
|---|---:|---|
| 이전 상승 또는 건설적 베이스 | 10 | 컵 형성 전 가격이 무질서한 급락만 있는 것이 아니라 회복 가능한 기반을 만든다. |
| 둥근 U자형 컵 구조 | 20 | V자 급반등보다 완만한 하락-바닥-회복 흐름이 보인다. |
| 적정 컵 깊이 | 15 | 림 대비 하락 폭이 과도하지 않고, 컵 저점이 구조를 훼손하지 않는다. |
| 오른쪽 림 회복 | 15 | 오른쪽 회복 구간이 이전 고점 부근까지 접근한다. |
| 손잡이 품질 | 15 | 손잡이 조정이 컵 상단부에서 얕고 짧게 진행된다. |
| 거래량 흐름 | 15 | 손잡이 구간 거래량은 줄고 돌파 구간에서 거래량이 증가한다. |
| 돌파 확인 | 10 | 손잡이 상단 또는 림 가격을 종가 기준으로 돌파한다. |

합계: 100점

### 감점 항목

| 항목 | 감점 |
|---|---:|
| 손잡이가 컵 중간 이하로 깊게 하락 | -20 |
| 컵이 둥글지 않고 V자 급반등에 가까움 | -15 |
| 오른쪽 림 회복이 약함 | -10 |
| 돌파 구간 거래량 증가가 없음 | -10 |
| 전고점 돌파 직후 긴 윗꼬리로 실패 | -10 |

### 앱 반영 상태

- `patterns.definition.scorecard`에 컵앤핸들 scorecard 저장
- `questions.rule_score`를 `pattern_score`로 API 응답에 노출
- 패턴 목록에서 컵앤핸들 기준 점수 표시
- 문제/연속 훈련/결과 화면에서 현재 문제의 패턴 적합도 점수 표시
